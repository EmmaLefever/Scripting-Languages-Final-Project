# Scripting-Languages-Final-Project
This repository contains the datasets and associated Jupyter notebook for my analysis of how economic factors influence fertility rates and life expectancy across countries of varying income levels. 

## Files:
- `GDP and GDP Per Capita.csv`: the GDP dataset, retrieved from [UNdata](https://data.un.org/).
- `Population Growth, Fertility and Mortality Indicators`: The population dataset, retrieved from [UNdata](https://data.un.org/).
- `Scripting_FinalProject_EmmaLefever.ipynb`: The Jupyter notebook which contains the data analysis, results, and interpretations.
